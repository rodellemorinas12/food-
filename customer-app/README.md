# Riders Local - Food Delivery App

A food delivery mobile application built with Flutter for Teresa Rizal, Philippines.

## Production Ready

This app has been configured for production deployment. See [PRODUCTION_DEPLOYMENT.md](PRODUCTION_DEPLOYMENT.md) for deployment instructions.

## Quick Start

1. Install dependencies:
   ```bash
   flutter pub get
   ```

2. Run in development mode:
   ```bash
   flutter run
   ```

3. Build for production:
   ```bash
   flutter build apk --release
   ```

## Features

- Browse local restaurants
- View restaurant menus
- Add items to cart
- Admin panel for restaurant and menu management
- Cash on delivery support

## API Requirements

- Node.js/Express backend
- MySQL database
- See `food_delivery_api/` directory for API code
